# Meet Assist — Project Instructions

## What This Is

Live meeting transcription (Zoom, etc.) using Deepgram's speech-to-text API. Runs on MacBook only (M4 Silicon for local audio capture).

## Runtime Environment

- **Machine:** MacBook (M4 Silicon) — NOT the iMac
- **API:** Deepgram (`$DEEPGRAM_API_KEY`)
- **Language:** TypeScript
